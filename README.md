## 🧠 Dynamic Programming Mastery

Solved and implemented various important **Dynamic Programming (DP) patterns** to strengthen problem-solving skills and understand optimal substructure and overlapping subproblems.

### Patterns Covered

- 0/1 Knapsack
- Unbounded Knapsack
- Longest Common Subsequence (LCS)
- LCS Variations (Longest Common Substring, Print LCS, etc.)
- Matrix Chain Multiplication (MCM)
- Tree Dynamic Programming

### Approaches Used

- Recursion
- Memoization (Top-Down DP)
- Tabulation (Bottom-Up DP)

### Language Used

- **C++**

These problems helped in building a strong foundation in **Dynamic Programming**, which is widely used in coding interviews and competitive programming.
